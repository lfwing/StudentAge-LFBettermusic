BetterMusic 公开 Mod 作者模板（1.5.1）

把整个 Bettermusic 文件夹放到你的 Mod 根目录：

<你的Mod根目录>/
└─ Bettermusic/
   ├─ Music/
   ├─ Lyrics/
   └─ Bettermusic.json

上传 Steam 创意工坊后，玩家无需把音乐、歌词或 JSON 复制到 BepInEx。
BetterMusic 主插件会扫描 StudentAge 的 Steam Workshop 安装目录并直接读取。

可用 EFFECT：
1. 1163,0
   停止并刷新插件音乐通道。

2. 1163,1,x,y,z,m[,u[,v]]
   单 Talk 背景型音乐。x=音乐ID；y=1/2/3；z=-1或1~4；m=0~12。

3. 1163,2,x,y,z,id1,id2...,-1[,u[,v]]
   单 Talk 唱歌型音乐。y 强制为3；z=1~4；-1 结束演唱角色表。

4. 1163,10,x,y,z,m[,u[,v]]
   跨 Talk 背景型音乐。y 强制为2，不阻止自动或快进。

5. 1163,20,x,y,z,id1,id2...,-1[,u[,v]]
   跨 Talk 唱歌型音乐。y 强制为1；播放期间阻止自动推进，但允许快进。

6. 1163,99,1 / 1163,99,2
   暂停当前插件音乐并记录位置 / 从记录位置恢复。

u/v：
- u=0 表示从音频开头播放；u>=1 表示从第 u 条有效时间歌词开始。
- v 为包含式终点；循环模式会在第 v 句结束后回到 u，单次模式会在此结束。
- JSON 元数据等无时间标签行不计入歌词句数。

扩展 LRC：
- id1[00:12.34]歌词：id1 对应 EFFECT 角色列表中的第1个角色。
- 使用“原文|译文”制作双语歌词。
- 唱歌模式中，男性歌词为天蓝色、女性为粉色、未匹配或无 idN 为“合唱”紫色。

重要：
1. JSON 中的相对路径以当前 Mod 自己的 Bettermusic 文件夹为基准。
2. 1163001 是插件代码级保留 ID，控制台名称固定为“校验音乐”。
3. 不同公开 Mod 应使用不冲突的音乐 ID。
4. z：1=默认、2=1.2倍、3=1.5倍、4=1.8倍；背景型音乐还允许 -1 隐藏歌词。
